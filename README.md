# 🟢 UniCrypto | Absolute Zero Protocol 

这是一个基于 **Cyberpunk 2077** 和 **高端金融科技 (FinTech)** 风格设计的加密货币登陆页面。该页面采用了极简主义的配色方案（Emerald Neon & Black），并加入了大量的动态视觉交互，旨在营造出一种“系统终端”的沉浸感。

## 🚀 核心特性

- **视觉效果**:
  - **动态扫描线 (CRT Scanlines)**: 模拟老式显示器的扫描质感。
  - **霓虹垂直轴 (Dynamic Neon Axis)**: 中轴线随鼠标横向位移，增加空间深度感。
  - **旋转 Motif 环**: 双重圆环旋转动画，象征去中心化网络。
  - **镜头光晕 (Lens Flare)**: 周期性闪烁的侧光，模拟电影镜头感。
- **交互设计**:
  - **打字机启动动画**: 进入页面时自动模拟数据加载过程。
  - **鼠标跟随反馈**: 页面背景和核心组件对鼠标移动做出微妙响应。
- **技术栈**:
  - HTML5 / CSS3 (原生动画)
  - **Tailwind CSS**: 快速排版与响应式布局。
  - **Google Fonts**: 使用 `Space Grotesk` 与 `Syncopate` 增强科技感。

## 🛠️ 安装与使用

由于该项目仅依赖于 CDN 资源，你无需安装任何依赖即可运行：

1. 下载或复制代码库中的 `index.html`。
2. 在浏览器中打开即可查看效果。
3. 如果需要自定义配色，可以在 CSS 根变量 `:root` 中修改 `--neon-emerald`。

## 📂 项目结构

- `index.html`: 包含所有的 HTML 结构、Tailwind 配置、内联 CSS 动画以及轻量级 JavaScript 交互逻辑。
